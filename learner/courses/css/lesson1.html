What is CSS?

CSS (Cascading Style Sheets) is the language used to style and layout web pages. While HTML provides the structure and content, CSS controls how that content looks — colors, fonts, spacing, layouts, and even animations. Think of HTML as the skeleton of a webpage and CSS as the skin and clothing that makes it look good.

CSS vs HTML: Understanding the Difference

HTML and CSS work together but serve different purposes:

HTML does:
  • Define what elements are on the page (headings, paragraphs, buttons)
  • Provide structure and meaning (semantics)
  • Contain the actual content (text, images)
CSS does:
  • Control visual appearance (colors, fonts, sizes)
  • Manage layout and positioning
  • Add visual effects (shadows, animations)

The Three Ways to Add CSS

There are three methods to apply CSS to your HTML, each with different use cases.

1. Inline CSS

Write CSS directly in an HTML element's style attribute. Use this sparingly — it's hard to maintain.




  This paragraph has inline styles.
Line by line:
  • style="..." — The style attribute holds CSS declarations
  • color: purple; — Sets text color to purple
  • font-size: 18px; — Sets text size to 18 pixels
  • Multiple declarations separated by semicolons
When to use inline CSS: Quick tests, email HTML, dynamic styles from JavaScript. Avoid for regular styling — it's hard to maintain and override.

Cascade order (lowest to highest priority):
  1. Browser defaults: Built-in styles
  2. External CSS: Linked stylesheets
  3. Internal CSS: Style tags in head
  4. Inline CSS: Style attribute on element
  5. !important: Overrides everything (use sparingly!)

Your First CSS Styles

Let's write some basic CSS to style a simple webpage:

/* styles.css - Your first stylesheet */

/* Style the whole page */
body {
  background-color: #ffffff;        /* White background */
  color: #333333;                   /* Dark gray text */
  font-family: Arial, sans-serif;   /* Clean, readable font */
  margin: 0;                        /* Remove default margin */
  padding: 20px;                    /* Add space inside */
}

/* Style headings */
h1 {
  color: #7C3AED;                   /* Purple color */
  font-size: 32px;                  /* Larger text */
  margin-bottom: 10px;              /* Space below */
}

h2 {
  color: #9333EA;                   /* Darker purple */
  font-size: 24px;
}

/* Style paragraphs */
p {
  font-size: 16px;
  line-height: 1.6;                 /* Better readability */
  margin-bottom: 15px;
}

/* Style links */
a {
  color: #7C3AED;
  text-decoration: none;            /* Remove underline */
}

a:hover {
  text-decoration: underline;       /* Underline on hover */
}
Common CSS properties used:
  • background-color — Background color of element
  • color — Text color
  • font-family — Type of font
  • font-size — Size of text
  • margin — Space outside element
  • padding — Space inside element

📚 Key Takeaways

  • CSS (Cascading Style Sheets) controls how HTML looks
  • Three ways to add CSS: inline, internal, external (recommended)
  • CSS syntax: selector { property: value; }
  • The cascade determines which styles take priority
  • External CSS is best for maintainability and caching
  • Always use semicolons after each declaration
