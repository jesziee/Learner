Lesson 8: CSS Grid

Master two-dimensional layouts with CSS Grid

Lesson 8 of 12

What is CSS Grid?

CSS Grid is a powerful layout system for creating two-dimensional layouts — handling both rows and columns simultaneously. Unlike Flexbox which is one-dimensional, Grid excels at creating complex page layouts, card grids, and any design that requires precise control over both axes. Grid and Flexbox complement each other and are often used together.

1. Grid Container and Items

Grid follows the same parent-child relationship as Flexbox.

/* Enable Grid on a container */
.container {
  display: grid;
}

/* The container becomes a grid container */
/* All direct children become grid items */

/* Grid terminology: */
/* - Grid container: Parent with display: grid */
/* - Grid items: Direct children */
/* - Grid lines: Horizontal and vertical lines */
/* - Grid tracks: Rows and columns */
/* - Grid cells: Intersection of row and column */
/* - Grid areas: Group of cells */

2. Defining Rows and Columns

grid-template-rows and grid-template-columns define the grid structure.

/* Define columns */
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
}
/* Creates 3 columns, each 200px wide */

/* Define rows */
.container {
  display: grid;
  grid-template-rows: 100px 200px;
}
/* Creates 2 rows: 100px and 200px tall */

/* Both together */
.container {
  display: grid;
  grid-template-columns: 200px 200px 200px;
  grid-template-rows: 100px 100px;
}
/* 3x2 grid (3 columns, 2 rows) */

/* Using fr unit (fraction of available space) */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
}
/* 3 equal columns, each taking 1/3 of space */

/* Mixed units */
.container {
  display: grid;
  grid-template-columns: 250px 1fr 2fr;
}
/* Fixed 250px, then 1/3 and 2/3 of remaining space */

/* Using repeat() function */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
/* Same as: 1fr 1fr 1fr */

.container {
  display: grid;
  grid-template-columns: repeat(4, 100px 50px);
}
/* Creates: 100px 50px 100px 50px 100px 50px 100px 50px */

/* Auto-sized columns */
.container {
  display: grid;
  grid-template-columns: auto auto auto;
}
/* Columns sized to content */

/* Mix of auto and fixed */
.container {
  display: grid;
  grid-template-columns: 200px auto 200px;
}
/* Fixed sidebars, fluid center */
Grid units:
  • px, em, rem — Fixed units
  • % — Percentage of container
  • fr — Fraction of available space
  • auto — Sized to content
  • min-content — Minimum content size
  • max-content — Maximum content size

3. Grid Gap

gap adds spacing between grid tracks.

/* Gap between rows and columns */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;              /* Gap between rows and columns */
}

/* Separate gaps */
.container {
  display: grid;
  gap: 20px 30px;        /* Row gap, column gap */
  /* OR */
  row-gap: 20px;
  column-gap: 30px;
}

/* Old syntax (still works) */
.container {
  grid-gap: 20px;
  grid-row-gap: 20px;
  grid-column-gap: 30px;
}

4. Placing Items in Grid

Control where items go using line-based placement.

/* Grid lines are numbered starting at 1 */

/* Place item by line numbers */
.item {
  grid-column-start: 1;
  grid-column-end: 3;
  /* OR shorthand */
  grid-column: 1 / 3;    /* Start at line 1, end at line 3 */
}

.item {
  grid-row: 1 / 2;       /* Row 1 only */
}

/* Span multiple cells */
.item {
  grid-column: 1 / span 2;    /* Start at 1, span 2 columns */
  grid-row: span 2;           /* Span 2 rows from current position */
}

/* Fill available space */
.full-width-item {
  grid-column: 1 / -1;    /* From first to last line (-1 = last line) */
}

/* Example: Grid with spanning header */
.layout {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

.header {
  grid-column: 1 / -1;    /* Full width */
}

.sidebar {
  grid-row: 2 / 4;        /* Spans 2 rows */
}

/* Result:
   ┌─────────────────────────────┐
   │           header            │
   ├────────┬────────────────────┤
   │        │                    │
   │ sidebar│      content       │
   │        │                    │
   ├────────┴────────────────────┤
   │           footer            │
   └─────────────────────────────┘
*/
Line numbering:
  • Lines start at 1 from left/top
  • Negative numbers count from end (-1 = last line)
  • span N spans N tracks

5. Grid Template Areas

Name grid areas for intuitive layout definition.

/* Define named areas */
.layout {
  display: grid;
  grid-template-columns: 200px 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
}

/* Assign items to areas */
.header { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main { grid-area: main; }
.footer { grid-area: footer; }

/* Result:
   ┌─────────────────────────┐
   │         header          │
   ├─────────┬───────────────┤
   │ sidebar │     main      │
   ├─────────┴───────────────┤
   │         footer          │
   └─────────────────────────┘
*/

/* Empty cells with . or none */
.layout {
  grid-template-areas:
    "header header header"
    "sidebar . main"
    "footer footer footer";
}

/* Responsive with media queries */
.layout {
  grid-template-areas:
    "header"
    "main"
    "sidebar"
    "footer";
}

@media (min-width: 768px) {
  .layout {
    grid-template-columns: 200px 1fr;
    grid-template-areas:
      "header header"
      "sidebar main"
      "footer footer";
  }
}
Why grid-template-areas: It provides a visual representation of your layout right in the CSS. Easy to understand and modify.

6. Auto-fit and Auto-fill

Create responsive grids without media queries.

/* auto-fill: Fill container with as many columns as fit */
.container {
  display: grid;
  grid-template-columns: repeat(auto-fill, 200px);
  gap: 20px;
}
/* Creates as many 200px columns as fit */

/* auto-fit: Same as auto-fill but collapses empty tracks */
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, 200px);
  gap: 20px;
}

/* With minmax for truly responsive grid */
.container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 20px;
}
/* Each column is at least 250px, at most 1fr */
/* Columns are created/removed based on available space */

/* Difference between auto-fill and auto-fit: */
/* auto-fill: Keeps empty tracks if items don't fill them */
/* auto-fit: Collapses empty tracks */

/* Responsive card grid - no media queries needed! */
.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 24px;
}

/* This single line creates a fully responsive grid */
/* - Cards are minimum 300px wide */
/* - Cards expand to fill available space */
/* - Columns are added/removed automatically */
minmax(min, max):
  • minmax(200px, 1fr) — At least 200px, at most 1fr
  • minmax(100px, auto) — At least 100px, expand to content
  • minmax(auto, 500px) — Content size, max 500px

7. Grid Alignment

Control alignment of the grid and items within cells.

/* Align the entire grid within container */
.container {
  display: grid;
  justify-content: center;    /* Center grid horizontally */
  align-content: center;      /* Center grid vertically */
}

/* Align all items within their cells */
.container {
  display: grid;
  justify-items: center;      /* Center items horizontally in cells */
  align-items: center;        /* Center items vertically in cells */
}

/* Place items shorthand */
.container {
  place-items: center;        /* justify-items + align-items */
}

/* Individual item alignment */
.item {
  justify-self: start;        /* Override container alignment */
  align-self: end;
}

/* Example: Centering content in grid cells */
.card-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
  align-items: stretch;       /* Default: items stretch to cell height */
}

.card {
  /* Cards fill their cells */
}

/* Example: Card with centered content */
.card {
  display: grid;
  place-items: center;
  min-height: 200px;
}
Alignment properties:
  • justify-content — Align grid horizontally in container
  • align-content — Align grid vertically in container
  • justify-items — Align all items horizontally in cells
  • align-items — Align all items vertically in cells

8. Implicit Grid

Grid automatically creates rows/columns for extra items.

/* Explicit grid: What you define */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 100px;
}
/* 3 columns, 2 rows defined (6 cells) */

/* Implicit grid: Auto-created for extra items */
/* If you have 9 items, Grid creates a 3rd row automatically */

/* Control implicit track sizes */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 150px;      /* Implicit rows are 150px */
}

/* minmax for auto rows */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: minmax(100px, auto);
}
/* Rows at least 100px, expand to content if needed */

/* Auto-placement direction */
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: row;        /* Default: fill row by row */
  grid-auto-flow: column;     /* Fill column by column */
  grid-auto-flow: dense;      /* Fill gaps with smaller items */
}

9. Practical Grid Patterns

Full Page Layout

/* Classic page layout */
.page {
  display: grid;
  grid-template-areas:
    "header header header"
    "nav main aside"
    "footer footer footer";
  grid-template-columns: 200px 1fr 150px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header { grid-area: header; }
.nav { grid-area: nav; }
.main { grid-area: main; }
.aside { grid-area: aside; }
.footer { grid-area: footer; }

Photo Gallery

/* Responsive image grid */
.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
  gap: 10px;
}

.gallery img {
  width: 100%;
  height: 200px;
  object-fit: cover;
}

/* Featured images span multiple cells */
.gallery .featured {
  grid-column: span 2;
  grid-row: span 2;
}

Dashboard Layout

/* Dashboard with varied cell sizes */
.dashboard {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: 100px;
  gap: 20px;
}

.widget-small {
  /* Default 1x1 */
}

.widget-wide {
  grid-column: span 2;
}

.widget-tall {
  grid-row: span 2;
}

.widget-large {
  grid-column: span 2;
  grid-row: span 2;
}

Masonry-like Grid

/* Grid with dense auto-placement */
.masonry {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  grid-auto-rows: 10px;
  grid-auto-flow: dense;
}

.masonry-item.small { grid-row: span 20; }  /* 200px */
.masonry-item.medium { grid-row: span 30; } /* 300px */
.masonry-item.tall { grid-row: span 40; }   /* 400px */

10. Grid vs Flexbox

When to use which layout method?

/* Use FLEXBOX for:
   - One-dimensional layouts (row OR column)
   - Navigation bars
   - Centering content
   - Flexible item sizing
   - Content-driven layouts
*/

/* Use GRID for:
   - Two-dimensional layouts (rows AND columns)
   - Page layouts
   - Card grids
   - Gallery layouts
   - Precise control over positioning
   - Overlapping content
*/

/* They work great together! */
.page {
  display: grid;
  grid-template-areas:
    "header"
    "main"
    "footer";
}

.header {
  display: flex;    /* Flexbox for navigation */
  justify-content: space-between;
  align-items: center;
}

.card-grid {
  display: grid;    /* Grid for cards */
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 20px;
}

.card {
  display: flex;    /* Flexbox inside cards */
  flex-direction: column;
}
Rule of thumb: Use Grid for the big picture (page layout, grid of items). Use Flexbox for components (navigation, buttons, card content).

📚 Key Takeaways

  • Grid is for two-dimensional layouts (rows AND columns)
  • grid-template-columns/rows define the grid structure
  • fr unit creates flexible fractions of available space
  • gap adds spacing between tracks
  • grid-template-areas names areas for intuitive layouts
  • auto-fit/auto-fill with minmax creates responsive grids
  • Items can span multiple cells using grid-column/row
  • Grid and Flexbox work great together
