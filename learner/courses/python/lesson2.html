Lesson 2: Variables & Data Types

Understand Python variables, strings, numbers, and basic data types

What are Variables?

Variables are containers for storing data values. In Python, you don't need to declare variable types - Python figures it out automatically. This is called dynamic typing. A variable is created the moment you assign a value to it using the equals sign (=).

Creating Variables

Creating variables in Python is straightforward. Just give your variable a name and assign a value using the assignment operator.

Variable Assignment:

# Creating variables is simple - just assign a value
name = "Alice"
age = 25
height = 5.6
is_student = True

# Python automatically knows the type
print(type(name))      # 
print(type(age))       # 
print(type(height))    # 
print(type(is_student)) # 

# Variables can be reassigned to different types
x = 10
print(x)       # 10
x = "hello"
print(x)       # hello

# Multiple assignment
a, b, c = 1, 2, 3
print(a, b, c) # 1 2 3

# Assign same value to multiple variables
x = y = z = 0
print(x, y, z) # 0 0 0

Lines 2-5: Create variables by assigning values. No type declaration needed.

Lines 8-11: The type() function shows what data type a variable contains.

Lines 14-17: Python variables can change types dynamically (though this isn't always good practice).

Lines 20-21: Multiple variables can be assigned in one line using tuple unpacking.

Lines 24-25: Chain assignment sets multiple variables to the same value.

💡 Variable Naming Rules

Variable names must start with a letter or underscore, can contain letters, numbers, and underscores, and are case-sensitive. Use descriptive names like user_age instead of x.

Numbers

Python has several numeric types. Integers are whole numbers, floats have decimal points, and complex numbers have real and imaginary parts. Python also supports arbitrary precision integers, meaning they can be as large as memory allows.

Numeric Types:

# Integers (whole numbers)
count = 42
temperature = -10
big_number = 1_000_000  # Underscores for readability

# Floats (decimal numbers)
price = 19.99
pi = 3.14159
scientific = 2.5e-3    # 0.0025 (scientific notation)

# Basic math operations
a = 10
b = 3

print(a + b)   # Addition: 13
print(a - b)   # Subtraction: 7
print(a * b)   # Multiplication: 30
print(a / b)   # Division: 3.333... (always returns float)
print(a // b)  # Floor division: 3 (rounds down)
print(a % b)   # Modulo: 1 (remainder)
print(a ** b)  # Exponentiation: 1000

# Type conversion
x = 5.7
print(int(x))    # 5 (truncates decimal)
print(float(10)) # 10.0

Lines 2-5: Integers can be positive, negative, or use underscores for readability.

Lines 8-10: Floats include decimal numbers and support scientific notation.

Lines 17-23: Python supports all standard math operations.

Line 20: Regular division always returns a float, even with integer operands.

Line 21: Floor division (//) rounds down to the nearest integer.

Lines 26-28: Convert between numeric types using int() and float().

Strings

Strings are sequences of characters enclosed in quotes. Python strings are immutable - once created, they cannot be changed. However, you can create new strings based on existing ones.

String Basics:

# Strings can use single or double quotes
greeting = "Hello"
name = 'Alice'

# Multi-line strings use triple quotes
bio = """
This is a multi-line string.
It can span multiple lines.
"""

# String operations
first = "Hello"
last = "World"
full = first + " " + last  # Concatenation
print(full)  # Hello World

# String repetition
print("=" * 10)  # ==========

# String indexing (0-based)
word = "Python"
print(word[0])   # P (first character)
print(word[-1])  # n (last character)
print(word[-2])  # o (second to last)

# String slicing
print(word[0:3])   # Pyt (characters 0, 1, 2)
print(word[2:])    # thon (from index 2 to end)
print(word[:4])    # Pyth (from start to index 3)
print(word[::2])   # Pto (every 2nd character)

Lines 2-3: Single and double quotes work identically for strings.

Lines 6-10: Triple quotes allow strings to span multiple lines.

Lines 13-16: Strings can be concatenated with the + operator.

Line 19: The * operator repeats strings.

Lines 22-25: Access characters by index. Negative indices count from the end.

Lines 28-31: Slicing extracts substrings using [start:end:step] syntax.

String Methods:

text = "  Hello, Python World!  "

# Case methods
print(text.upper())      # "  HELLO, PYTHON WORLD!  "
print(text.lower())      # "  hello, python world!  "
print(text.title())      # "  Hello, Python World!  "
print(text.strip())      # "Hello, Python World!" (removes whitespace)

# Search methods
sentence = "Python is awesome"
print(sentence.find("is"))     # 7 (index of substring)
print(sentence.startswith("Python"))  # True
print(sentence.endswith("awesome"))   # True

# Replace and split
print(sentence.replace("awesome", "amazing"))  # Python is amazing
print(sentence.split())        # ['Python', 'is', 'awesome']
print("a,b,c".split(","))     # ['a', 'b', 'c']

# Check contents
print("hello".isalpha())    # True (only letters)
print("123".isdigit())      # True (only digits)
print("hello123".isalnum()) # True (letters and digits)

Lines 4-7: Case conversion methods return new strings with changed case.

Line 7: strip() removes leading and trailing whitespace.

Lines 10-14: Search for substrings and check string boundaries.

Lines 17-19: Replace text or split strings into lists.

Lines 22-24: Check if string contains only certain character types.

String Formatting

Python offers several ways to format strings with variable values. F-strings (formatted string literals) are the modern and recommended approach.

String Formatting Methods:

name = "Alice"
age = 25
score = 95.678

# f-strings (Python 3.6+) - Recommended
print(f"Name: {name}, Age: {age}")
print(f"Score: {score:.2f}")  # Score: 95.68 (2 decimal places)
print(f"Next year: {age + 1}")  # Expressions work too!

# format() method
print("Name: {}, Age: {}".format(name, age))
print("Name: {0}, Age: {1}, Again: {0}".format(name, age))

# %-formatting (old style)
print("Name: %s, Age: %d" % (name, age))
print("Score: %.2f" % score)

# Formatting options
price = 1234.5
print(f"${price:,.2f}")    # $1,234.50 (comma separators)
print(f"{name:>10}")       # "     Alice" (right-aligned, 10 chars)
print(f"{name:<10}")       # "Alice     " (left-aligned)
print(f"{name:^10}")       # "  Alice   " (centered)

Lines 6-9: F-strings use f"..." with variables in curly braces. They support formatting and expressions.

Line 8: Format specifiers like :.2f control decimal places.

Lines 12-13: The format() method is useful for templates.

Lines 16-17: The old %-formatting is still seen in legacy code.

Lines 20-23: Format specifiers for alignment, commas, and width.

💡 Pro Tip: F-strings

F-strings are faster and more readable than other formatting methods. Always prefer them unless you need to support Python versions older than 3.6.

Booleans

Booleans represent one of two values: True or False. They're essential for making decisions in your code and are the result of comparison operations.

Boolean Values:

# Boolean literals
is_active = True
is_admin = False

# Comparisons return booleans
print(5 > 3)    # True
print(5 < 3)    # False
print(5 == 5)   # True (equality)
print(5 != 3)   # True (not equal)
print(5 >= 5)   # True (greater or equal)

# Truthy and Falsy values
# These values are "falsy" (evaluate to False in boolean context):
# - None, False, 0, 0.0, "", [], {}, ()

bool(0)        # False
bool("")       # False
bool([])       # False
bool(None)     # False

# Everything else is "truthy"
bool(1)        # True
bool("hello")  # True
bool([1, 2])   # True

# Logical operators
print(True and False)  # False (both must be True)
print(True or False)   # True (at least one True)
print(not True)        # False (inverts the value)

Lines 2-3: Boolean values are capitalized: True and False.

Lines 6-11: Comparison operators return boolean values.

Lines 14-24: Many values are implicitly truthy or falsy in boolean contexts.

Lines 27-29: Logical operators combine boolean values.

Type Conversion

You can convert between different data types using built-in functions. This is called type casting or type conversion.

Converting Types:

# String to number
age_str = "25"
age_int = int(age_str)
age_float = float(age_str)
print(age_int + 5)    # 30

# Number to string
num = 42
num_str = str(num)
print("The answer is " + num_str)  # The answer is 42

# To boolean
print(bool(1))    # True
print(bool(0))    # False
print(bool(""))   # False

# Handling conversion errors
try:
    bad = int("hello")  # This will fail
except ValueError:
    print("Cannot convert to integer")

# input() always returns a string
user_input = input("Enter a number: ")
number = int(user_input)  # Convert to integer
print(number * 2)

Lines 2-6: Convert strings to numbers with int() and float().

Lines 9-11: Convert anything to a string with str().

Lines 14-16: Convert to boolean with bool().

Lines 19-22: Invalid conversions raise ValueError - handle with try/except.

Lines 25-27: Always convert input() results if you need numbers.
