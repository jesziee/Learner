Lesson 1: Introduction to Python

Learn what Python is, why it's popular, and write your first Python program

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, Python has become one of the most popular programming languages in the world. Its clean syntax and emphasis on readability make it an excellent choice for beginners while remaining powerful enough for experts.

Python is used in web development, data science, artificial intelligence, automation, scientific computing, and much more. Companies like Google, Netflix, Instagram, and Spotify use Python extensively in their technology stacks.

Why Learn Python?

Python offers several advantages that make it an excellent first programming language. Its syntax resembles natural English, making it easier to read and write. The extensive standard library provides pre-built modules for countless tasks, and the active community means help is always available. Python's versatility allows you to build almost anything, from simple scripts to complex machine learning models.

Installing Python

Before we start coding, you need to install Python on your computer. Python comes pre-installed on most Linux and macOS systems. On Windows, you'll need to download it from the official Python website.

Checking if Python is installed:

# Open your terminal or command prompt and type:
python --version

# Or on some systems:
python3 --version

# Output should be something like:
# Python 3.11.0

Line 2: This command checks the default Python version on your system.

Line 5: On many systems, Python 3 is installed as python3 to distinguish from older Python 2.

Line 8: The output shows your installed Python version. Make sure you have Python 3.6 or newer.

💡 Tip: Python Virtual Environments

When working on Python projects, it's best practice to create virtual environments to isolate dependencies. Use python -m venv myenv to create one.

Your First Python Program

Let's write the traditional "Hello, World!" program. This simple program introduces you to Python's print function, which displays output to the screen.

Hello World:

# This is a comment - Python ignores everything after #
# Comments help explain what your code does

print("Hello, World!")

# You can also use single quotes
print('Welcome to Python!')

# Multiple items are separated by spaces
print("Hello", "Python", "Learner!")

Lines 1-2: Comments start with #. They're ignored by Python but help humans understand the code.

Line 4: The print() function displays text to the screen. Text (strings) must be in quotes.

Line 7: Python accepts both single (') and double (") quotes for strings.

Line 10: Multiple arguments to print are separated by spaces by default.

The Python Interpreter

Python is an interpreted language, meaning you can run code line by line in an interactive mode called the REPL (Read-Eval-Print Loop). This is great for testing and learning.

Using the Python REPL:

# Start the REPL by typing python or python3 in your terminal
# You'll see >>> which means Python is waiting for input

>>> 2 + 2
4

>>> "Hello" + " " + "World"
'Hello World'

>>> name = "Alice"
>>> print(f"Hello, {name}!")
Hello, Alice!

>>> exit()  # Type this to exit the REPL

Line 1: The >>> prompt indicates interactive mode is active.

Line 4: Python evaluates expressions immediately and shows the result.

Line 7: Strings can be concatenated with the + operator.

Line 10: Variables are created with assignment (=).

Line 11: f-strings allow embedding expressions in strings using curly braces.

Running Python Files

While the REPL is great for quick tests, most Python code is written in files with the .py extension. You can create these files in any text editor or IDE.

Running a Python file:

# Save this as hello.py
name = input("What is your name? ")
print(f"Nice to meet you, {name}!")

# Run it from terminal:
# python hello.py

Line 2: input() prompts the user for text input and returns what they type.

Line 3: The f-string inserts the variable value into the string.

Line 6: Run Python files by passing the filename to the python command.
