Lesson 4: Conditionals

Learn to make decisions in your code with if, elif, and else statements

What are Conditionals?

Conditional statements allow your program to make decisions and execute different code based on conditions. Using if, elif, and else, you can control the flow of your program and respond differently to various situations.

The if Statement

The if statement executes code only when a condition is true. Python uses indentation (typically 4 spaces) to define code blocks.

Basic if Statement:

# Basic if statement
age = 18

if age >= 18:
    print("You are an adult")
    print("You can vote!")

# Multiple conditions
temperature = 75

if temperature > 80:
    print("It's hot outside!")

if temperature < 60:
    print("It's cold outside!")

# Check if a number is positive
number = 42

if number > 0:
    print(f"{number} is positive")

# Check if a list is not empty
items = [1, 2, 3]

if items:  # Truthy check
    print(f"List has {len(items)} items")

Lines 4-6: The if statement checks a condition. If true, the indented block runs.

Lines 9-16: Multiple independent if statements can check different conditions.

Lines 19-21: The condition can be any expression that evaluates to a boolean.

Lines 24-27: Empty collections are falsy, non-empty ones are truthy.

💡 Python Indentation

Python uses indentation to define code blocks, not curly braces like other languages. Always use consistent indentation (4 spaces is standard) to avoid errors.

if-else Statement

The else clause provides an alternative path when the if condition is false. This creates a two-way decision structure.

if-else Structure:

# Basic if-else
age = 16

if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

# Check even or odd
number = 7

if number % 2 == 0:
    print(f"{number} is even")
else:
    print(f"{number} is odd")

# Password check
password = "secret123"

if len(password) >= 8:
    print("Password accepted")
else:
    print("Password too short (minimum 8 characters)")

# Check if user is logged in
user = None

if user:
    print(f"Welcome back, {user}!")
else:
    print("Please log in")

Lines 4-7: The else block runs when the if condition is false.

Lines 10-15: Use modulo operator to check even/odd numbers.

Lines 18-22: Practical example of validating input.

Lines 25-29: None is falsy, so this checks if user exists.

if-elif-else Statement

Use elif (else if) to check multiple conditions in sequence. Python checks each condition in order and runs the first one that's true.

Multiple Conditions:

# Grade calculator
score = 85

if score >= 90:
    grade = "A"
elif score >= 80:
    grade = "B"
elif score >= 70:
    grade = "C"
elif score >= 60:
    grade = "D"
else:
    grade = "F"

print(f"Score: {score}, Grade: {grade}")

# Time of day greeting
hour = 14  # 24-hour format

if hour < 12:
    greeting = "Good morning"
elif hour < 17:
    greeting = "Good afternoon"
elif hour < 21:
    greeting = "Good evening"
else:
    greeting = "Good night"

print(greeting)

# Age categories
age = 25

if age < 13:
    category = "Child"
elif age < 20:
    category = "Teenager"
elif age < 65:
    category = "Adult"
else:
    category = "Senior"

print(f"You are a {category}")

Lines 4-14: elif allows checking multiple conditions. Only the first true block runs.

Lines 17-28: Conditions are checked top-to-bottom. Order matters!

Lines 31-41: Complex multi-way decisions are easy with elif chains.

💡 Important: Order Matters

Python evaluates conditions from top to bottom and stops at the first true one. Make sure your conditions are in the right order for your logic to work correctly.

Nested Conditionals

You can place if statements inside other if statements. This is called nesting and allows for more complex decision-making.

Nested if Statements:

# Login system with nested conditions
username = "admin"
password = "secret123"
is_admin = True

if username:
    if password:
        if username == "admin" and password == "secret123":
            if is_admin:
                print("Admin access granted")
            else:
                print("User access granted")
        else:
            print("Invalid credentials")
    else:
        print("Password required")
else:
    print("Username required")

# Nested vs flat - prefer flat when possible
age = 25
has_ticket = True

# Nested version (harder to read)
if age >= 18:
    if has_ticket:
        print("Welcome to the concert!")

# Flat version with 'and' (preferred)
if age >= 18 and has_ticket:
    print("Welcome to the concert!")

# Early return pattern (even better)
def check_access(age, has_ticket):
    if age < 18:
        return "Too young"
    if not has_ticket:
        return "No ticket"
    return "Welcome!"

Lines 6-19: Nested if statements create multiple levels of conditions.

Lines 22-34: Often, combining conditions with and is clearer than deep nesting.

Lines 37-42: The early return pattern avoids deep nesting and is more readable.

Logical Operators with Conditionals

Combine conditions using and, or, and not for more complex logic.

Combining Conditions:

# Using 'and' - both conditions must be true
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive")

# Using 'or' - at least one condition must be true
is_member = False
has_coupon = True

if is_member or has_coupon:
    print("You get a discount!")

# Using 'not' - inverts the condition
is_banned = False

if not is_banned:
    print("Access allowed")

# Complex conditions with parentheses
age = 25
is_student = False
has_id = True

# Parentheses clarify complex logic
if (age >= 18 and age <= 25) and (is_student or has_id):
    print("Youth discount applies!")

# Checking multiple values
day = "Saturday"

if day == "Saturday" or day == "Sunday":
    print("It's the weekend!")

# Better way using 'in'
if day in ["Saturday", "Sunday"]:
    print("It's the weekend!")

Lines 5-6: and requires both conditions to be true.

Lines 10-12: or requires at least one condition to be true.

Lines 15-17: not inverts the boolean value.

Lines 23-24: Use parentheses to clarify complex conditions.

Lines 32-34: Using in with a list is cleaner than multiple or conditions.

Conditional Expressions (Ternary Operator)

Python offers a concise way to write simple if-else statements in a single line using conditional expressions.

One-Line Conditionals:

# Traditional if-else
age = 20

if age >= 18:
    status = "adult"
else:
    status = "minor"

print(status)  # adult

# Conditional expression (ternary)
status = "adult" if age >= 18 else "minor"

# More examples
score = 75
result = "Pass" if score >= 60 else "Fail"
print(result)  # Pass

# In f-strings
print(f"You are an {'adult' if age >= 18 else 'minor'}")

# With function calls
def get_fee(is_member):
    return 0 if is_member else 10

print(get_fee(True))   # 0
print(get_fee(False))  # 10

# Nested conditional expressions (use sparingly)
score = 85
grade = "A" if score >= 90 else "B" if score >= 80 else "C"
print(grade)  # B

Lines 4-9: Traditional multi-line if-else statement.

Line 12: Same logic in one line: value_if_true if condition else value_if_false.

Lines 16-17: Useful for simple value assignments based on conditions.

Line 20: Can be used inside f-strings for dynamic text.

Lines 23-26: Works well with functions for cleaner code.

Lines 29-31: Nested ternary is possible but harder to read - avoid deep nesting.

Match Statement (Python 3.10+)

Python 3.10 introduced the match statement for pattern matching, which is powerful for comparing a value against multiple patterns.

Pattern Matching:

# Basic match statement
day = "Monday"

match day:
    case "Monday":
        print("Start of the work week")
    case "Friday":
        print("TGIF!")
    case "Saturday" | "Sunday":  # Multiple patterns
        print("It's the weekend!")
    case _:  # Default case (like 'else')
        print("It's a regular weekday")

# Matching with conditions (guards)
score = 85

match score:
    case s if s >= 90:
        grade = "A"
    case s if s >= 80:
        grade = "B"
    case s if s >= 70:
        grade = "C"
    case _:
        grade = "F"

print(f"Grade: {grade}")

# Matching data structures
point = (3, 4)

match point:
    case (0, 0):
        print("Origin")
    case (x, 0):
        print(f"On x-axis at {x}")
    case (0, y):
        print(f"On y-axis at {y}")
    case (x, y):
        print(f"Point at ({x}, {y})")

Lines 4-12: Basic match-case structure, similar to switch in other languages.

Line 9: Use | to match multiple patterns.

Line 11: _ is the wildcard pattern, matches anything.

Lines 15-26: Guards with if add conditions to patterns.

Lines 29-39: Match can destructure tuples and other data structures.

💡 Python Version

The match statement requires Python 3.10 or newer. For older versions, use if-elif-else chains instead.
