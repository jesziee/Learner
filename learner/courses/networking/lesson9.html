🛡️ Lesson 9: Firewalls & Security

Stateful firewalls, VPNs, IDS/IPS, zero trust - enterprise network protection

Firewall Fundamentals

Network security device filtering traffic by rules. Default deny.

Packet Filter: Stateless Stateful: Connection tracking Proxy: Application layer

Stateful Inspection

Tracks connection state table. Allows return traffic automatically.

State Table:

ConnID | Proto | SrcIP:Port | DstIP:Port | State
1     | TCP  | 10.0.1.2:54321 | 8.8.8.8:53 | ESTABLISHED
2     | UDP  | 192.168.1.10:123 | ntp.org:123 | RELATED

Sees SYN, SYN-ACK, tracks bidirectional.

VPN Protocols

Secure tunnels over untrusted networks.

IPsec: L3 encrypted tunnel
OpenVPN: SSL/TLS based
WireGuard: Modern, fast
PPTP:  Obsolete (insecure)

IDS vs IPS

IDS: Detects/alerts (passive)
IPS: Detects/blocks (inline)

Snort/Suricata: Popular open source.

Zero Trust Model

Never trust, always verify. Perimeter is dead.

  • Verify explicitly
  • Least privilege
  • Assume breach

Common Rules

# iptables example
-A INPUT -i lo -j ACCEPT
-A INPUT -p tcp --dport 22 -j ACCEPT
-A INPUT -m state --state ESTABLISHED -j ACCEPT  
-A INPUT -j DROP

Practice: Security Headers Test

HTTPS Security Check:

async function testSecurity() {
  const sites = ['https://httpbin.org/headers', 'https://google.com'];
  
  for (const site of sites) {
    try {
      const res = await fetch(site, {method: 'HEAD'});
      console.log(`✅ ${site.split('/')[2]}: HTTPS OK (${res.status})`);
      
      // Check common headers
      if (res.headers.get('strict-transport-security')) {
        console.log('  → HSTS enabled');
      }
    } catch(e) {
      console.log(`❌ ${site}: ${e.message}`);
    }
  }
}
testSecurity();

Key Takeaways

  • Stateful firewall tracks connections
  • VPN: IPsec/OpenVPN/WireGuard
  • IDS alerts, IPS blocks
  • Zero Trust: Verify always
  • Default deny policy essential
