🌐 Lesson 1: Networking Fundamentals

Understand what networks are, how they work, and why they're essential for modern computing - from beginner concepts to pro principles

What is a Computer Network?

A computer network is a collection of interconnected devices (computers, servers, smartphones) that communicate and share resources. Networks enable data exchange over short or long distances.

Analogy: Networks are like highways where data packets travel like cars between cities.

Why Networks Matter

Essential for:

Resource sharing Communication Cloud services Internet access

Network Types

LAN • MAN • WAN

LAN: Local network (office/home WiFi)
MAN: City-wide (campuses)
WAN: Global (Internet)

Key Devices

Core Network Hardware:

Routers  → Network gateways
Switches → LAN connections
Firewalls → Security
Modems   → Internet converter

Router: Traffic director

Switch: Local device connector

OSI Model

7 Layers:

7 App     HTTP
6 Present Encrypt
5 Session Connect
4 Trans   TCP/UDP
3 Net     IP route
2 Link    MAC/switch
1 Phys    Cables

Mnemonic: All People Seem To Need Data Processing

Packet Flow

1. Create → 2. Encapsulate → 3. Transmit → 4. Decapsulate → 5. Deliver

Practice

fetch('https://httpbin.org/ip')
  .then(res => res.json())
  .then(data => console.log('IP:', data.origin));

Key Takeaways

  • Networks connect & share
  • LAN/WAN, routers/switches
  • OSI 7 layers
  • Packets, not whole files
