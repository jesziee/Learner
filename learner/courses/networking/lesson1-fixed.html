🌐 Lesson 1: Networking Fundamentals

Understand what networks are, how they work, and why they're essential for modern computing - from beginner concepts to pro principles

What is a Computer Network?

A computer network is a collection of interconnected devices (computers, servers, smartphones) that communicate and share resources. Networks enable data exchange over short or long distances.

Analogy: Networks are like highways where data packets travel like cars between cities.

Why Networks Matter

Essential for modern computing:

File sharing Internet access Cloud services Communication Centralized management

Network Types

Networks by size:

LAN, MAN, WAN

LAN: Local (home/office WiFi)
MAN: City (campus networks)
WAN: Global (Internet)

Key Network Devices

Hardware that makes networks work:

Network Devices:

Router     → Connects networks (home ↔ Internet)
Switch     → Connects LAN devices
Firewall   → Security gateway
Access Point → WiFi hotspot
Modem      → Internet signal converter

Router: Directs traffic between networks.

Switch: High-speed LAN connections.

OSI Model (7 Layers)

Standard blueprint for network communication:

OSI Layers (Bottom → Top):

7 Application  ← HTTP, Email
6 Presentation ← Encryption
5 Session      ← Connections
4 Transport    ← TCP/UDP
3 Network      ← IP routing  
2 Data Link    ← MAC, switches
1 Physical     ← Cables, signals

Mnemonic: "All People Seem To Need Data Processing"

Data Packet Flow

Data broken into packets, routed independently:

  1. App creates message
  2. Layers add headers
  3. Packets travel network
  4. Destination reconstructs

Practice: Network Test

JS Network Checker:

async function testNetwork() {
  try {
    const res = await fetch('https://httpbin.org/ip');
    const data = await res.json();
    console.log('Network OK! IP:', data.origin);
  } catch {
    console.log('Network issue');
  }
}
testNetwork();

Key Takeaways

Core Concepts:

  • Networks connect & share resources
  • LAN/WAN types, router/switch devices
  • OSI 7-layer model
  • Data travels in packets
