📡 Lesson 4: WiFi & Wireless

Wireless networking standards, security, channels, interference - optimize your WiFi

WiFi Standards (IEEE 802.11)

Evolution of wireless LAN technology.

WiFi Generations:

802.11b: 11 Mbps (2.4GHz)
802.11g: 54 Mbps (2.4GHz) 
802.11n: 600 Mbps (2.4/5GHz)
802.11ac: 6.9 Gbps (5GHz)
802.11ax (WiFi6): 9.6 Gbps (2.4/5GHz)
WiFi6E: +6GHz band

Higher = faster + more devices.

WiFi Channels & Interference

2.4GHz (14 channels, 1/6/11 non-overlap), 5GHz (more channels).

Pro Tip: Use WiFi analyzer apps to pick best channel.

Wireless Security

Security Evolution:

WEP:  Cracked (avoid!)
WPA:  TKIP (weak)
WPA2: AES (standard)
WPA3: Enhanced (latest)

Access Points vs Routers

AP: Wireless extension only
Router: AP + NAT + Firewall + Switch

Mesh: Whole-home coverage MU-MIMO: Multiple devices

Wireless Topologies

Star (AP central), Mesh (self-healing), Point-to-Point bridges.

Practice: WiFi Scanner Simulation

Network Connection Test:

async function testWiFiQuality() {
  const tests = ['https://httpbin.org/delay/1', 'https://httpbin.org/ip'];
  let success = 0;
  
  for (const url of tests) {
    try {
      const start = performance.now();
      await fetch(url);
      const latency = performance.now() - start;
      console.log(`✅ ${url}: ${latency.toFixed(0)}ms`);
      success++;
    } catch(e) {
      console.log(`❌ ${url}: Failed`);
    }
  }
  
  const score = (success / tests.length) * 100;
  console.log(`WiFi Score: ${score}%`);
}
testWiFiQuality();

Key Takeaways

  • WiFi6 (802.11ax) = fastest/current standard
  • 2.4GHz range, 5GHz speed
  • WPA3 > WPA2 security
  • Channels 1/6/11 non-overlapping (2.4GHz)
  • Mesh for home coverage
