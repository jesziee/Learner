🔌 Lesson 3: Ethernet & LAN

Deep dive into wired networks: MAC addresses, switches, ARP, VLANs, cabling standards

Ethernet Basics

Standard for wired LAN (IEEE 802.3). Used in 90%+ of office/home networks.

Speeds: 10/100/1000/10G Mbps Cables: Cat5e/Cat6

MAC Addresses

48-bit hardware address burned into NIC (Network Interface Card).

MAC Format:

Format: AA:BB:CC:DD:EE:FF (hex)
Example: 00:1A:2B:3C:4D:5E
OUI (first 24 bits) = Vendor ID

Layer 2 addressing (local network only).

Switches vs Hubs

Hub: Dumb broadcast (collision domain)
Switch: Smart forwarding (learns MAC table)

MAC Table Example:
Port1: 00:11:22:33:44:55
Port2: AA:BB:CC:DD:EE:FF
Unknown → Flood all ports

ARP Protocol

Address Resolution: IP → MAC mapping.

ARP Flow:

1. Who has 192.168.1.10? (Broadcast)
2. 192.168.1.10 is at AA:BB:CC:DD:EE:FF (Unicast)
3. Cache result in ARP table

VLANs (Virtual LANs)

Logically segment physical switch into multiple networks.

VLAN 10: Finance, VLAN 20: HR (traffic isolation).

Cabling Standards

Cat5e: 1Gbps / 100m
Cat6:  10Gbps / 55m
Cat6a: 10Gbps / 100m
Fiber: 100Gbps+ unlimited

Practice: Network Info

Browser Network Details:

async function getNetworkInfo() {
  try {
    const res = await fetch('https://httpbin.org/ip');
    const ipData = await res.json();
    
    // Get local IPs (limited in browser)
    console.log('Public IP:', ipData.origin);
    console.log('Local IPs via RTCPeerConnection hack:');
    
    const pc = new RTCPeerConnection({iceServers:[]});
    pc.createDataChannel('');
    pc.createOffer().then(pc.setLocalDescription.bind(pc));
    pc.onicecandidate = (ice) => {
      if (ice.candidate) {
        const ipMatch = /([0-9]{1,3}(\.[0-9]{1,3}){3})/.exec(ice.candidate.candidate);
        if (ipMatch) console.log(ipMatch[1]);
      }
    };
  } catch(e) {
    console.log('Network info unavailable');
  }
}
getNetworkInfo();

Key Takeaways

  • MAC: 48-bit hardware address (AA:BB:CC:DD:EE:FF)
  • Switch learns MAC table, forwards smartly
  • ARP resolves IP→MAC
  • VLANs segment traffic
  • Cat6+ for Gigabit+ speeds
