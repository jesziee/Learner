🚀 Lesson 9: HTML5 New Elements

Modern HTML5 features and APIs

What is HTML5?

HTML5 is the latest version of HTML, introducing new semantic elements, form input types, multimedia support, and APIs for building modern web applications.

Key improvements: Better semantic structure, native multimedia support, new form input types, offline storage, geolocation, canvas for graphics, and drag-and-drop APIs.

Multimedia Elements

Embed video content without plugins like Flash.

// Basic video
<video src="video.mp4">video>

// Video with controls and multiple sources
<video controls width="640" height="360">
    <source src="video.mp4" type="video/mp4">
    <source src="video.webm" type="video/webm">
    Your browser doesn't support video.
video>

// Common attributes:
controls    // Show play/pause controls
autoplay    // Auto-play (often blocked without muted)
muted       // Mute audio
loop        // Loop video
poster="img.jpg"  // Thumbnail before play
preload="auto"   // Load behavior
playsinline // Play inline on mobile (not fullscreen)

Embed audio content.

// Basic audio
<audio src="audio.mp3" controls>audio>

// Audio with multiple sources
<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    <source src="audio.ogg" type="audio/ogg">
    Your browser doesn't support audio.
audio>

// Common attributes:
controls  // Show audio controls
autoplay  // Auto-play (often blocked)
muted     // Mute
loop      // Loop audio
preload   // Load behavior

Canvas and SVG

A drawable region for JavaScript graphics.

// Canvas element
<canvas id="myCanvas" width="400" height="300">
    Your browser doesn't support canvas.
canvas>

// JavaScript draws on canvas:
const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d');

// Draw a rectangle
ctx.fillStyle = '#7C3AED';
ctx.fillRect(10, 10, 100, 50);

// Uses: Games, charts, image editing, animations

Inline scalable vector graphics.

// Inline SVG
<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" fill="#7C3AED"/>
svg>

// SVG is scalable, can be styled with CSS,
// and can be manipulated with JavaScript

New Form Input Types

HTML5 added many new input types with built-in validation:

// Date and time inputs
<input type="date">           // Date picker
<input type="time">           // Time picker
<input type="datetime-local"> // Date and time
<input type="month">          // Month picker
<input type="week">           // Week picker

// Other new types
<input type="email">         // Email validation
<input type="url">           // URL validation
<input type="tel">           // Phone number
<input type="search">        // Search field
<input type="number">        // Number with spinner
<input type="range">         // Slider
<input type="color">         // Color picker

// New attributes
<input type="text" autocomplete="on">
<input type="text" autofocus>
<input type="text" placeholder="Hint">
<input type="text" required>
<input type="text" pattern="[A-Za-z]{3}">

HTML5 APIs Overview

HTML5 introduced JavaScript APIs for powerful web applications:

Local Storage

// Store data in the browser
localStorage.setItem('username', 'John');
const name = localStorage.getItem('username');
localStorage.removeItem('username');
localStorage.clear();  // Clear all

// sessionStorage works the same but clears when tab closes

Geolocation

// Get user's location
navigator.geolocation.getCurrentPosition(function(position) {
    console.log(position.coords.latitude);
    console.log(position.coords.longitude);
});
