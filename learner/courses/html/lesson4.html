🔗 Lesson 4: Links and Images

Adding hyperlinks and images to pages

Anchor Tags (Links)

The element creates hyperlinks that let users navigate to other pages, files, or locations within a page.

// Basic link syntax:
<a href="https://example.com">Click herea>
     ↑                      ↑
   attribute            link text (clickable)
   (destination)

// The href attribute specifies the destination URL

Types of Links

Absolute URLs

Full web addresses to external sites:

// Link to another website
<a href="https://www.google.com">Go to Googlea>

// Link to a specific page
<a href="https://example.com/about">About Usa>

Relative URLs

Links to pages within your own website:

// Same folder
<a href="about.html">Abouta>

// Subfolder
<a href="products/item.html">Producta>

// Parent folder (go up one level)
<a href="../index.html">Homea>

// Root-relative (starts from website root)
<a href="/contact.html">Contacta>

Page Anchors

Link to specific sections within the same page:

// Create a target with an id:
<h2 id="section1">Section 1h2>

// Link to that section:
<a href="#section1">Jump to Section 1a>

// Link to a section on another page:
<a href="page.html#section1">Section 1 on another pagea>

Link Targets

The target attribute controls where the link opens:

// Open in same tab (default)
<a href="https://example.com">Same taba>

// Open in new tab/window
<a href="https://example.com" target="_blank">New taba>

// Security best practice for external links:
<a href="https://example.com" 
   target="_blank" 
   rel="noopener noreferrer">
    Safe external link
a>

Security tip: Always add rel="noopener noreferrer" to external links with target="_blank". This prevents the linked page from accessing your page's window object.

Other Link Types

// Email link (opens email client)
<a href="mailto:someone@example.com">Send Emaila>

// Email with subject and body
<a href="mailto:someone@example.com?subject=Hello&body=Hi there">
    Send Email
a>

// Phone number (useful on mobile)
<a href="tel:+1234567890">Call Usa>

// Download link
<a href="files/document.pdf" download>Download PDFa>

// Download with custom filename
<a href="files/doc.pdf" download="my-document.pdf">Downloada>

Images

The element embeds images in your page. It's a self-closing tag.

// Basic image syntax:
<img src="photo.jpg" alt="Description of image">
     ↑                ↑
   source          alternative text
   (required)      (required for accessibility)

Required Attributes

  • src: The path to the image file (URL or file path)
  • alt: Alternative text describing the image (for accessibility and when image fails to load)

Alt text is crucial! Screen readers use it to describe images to visually impaired users. It also displays if the image fails to load and helps search engines understand the image content.

Image Sizing

// Set width and height (in pixels)
<img src="photo.jpg" 
     alt="A sunset" 
     width="300" 
     height="200">

// Or use CSS for responsive images
<img src="photo.jpg" 
     alt="A sunset" 
     style="width: 100%; max-width: 500px;">

Best practice: Always specify width and height attributes to prevent layout shift while the image loads. Use CSS for responsive sizing.

Image Paths

// Image in same folder as HTML file
<img src="photo.jpg" alt="Photo">

// Image in a subfolder
<img src="images/photo.jpg" alt="Photo">

// Image in parent folder
<img src="../photo.jpg" alt="Photo">

// Image from another website (absolute URL)
<img src="https://example.com/images/photo.jpg" alt="Photo">

Image Formats

Common image formats for the web:

  • JPEG (.jpg): Photos and complex images with many colors
  • PNG (.png): Images with transparency, logos, icons
  • GIF (.gif): Simple animations, limited colors
  • WebP (.webp): Modern format with better compression
  • SVG (.svg): Vector graphics, logos, icons (scales perfectly)

Images as Links

Wrap an image in an anchor tag to make it clickable:

// Clickable image
<a href="https://example.com">
    <img src="logo.png" alt="Company Logo">
a>

// Common for logos that link to homepage

Figure and Figcaption

Semantic elements for images with captions:

<figure>
    <img src="sunset.jpg" alt="A beautiful sunset over the ocean">
    <figcaption>Photo taken at Malibu Beach, Californiafigcaption>
figure>

//

Key Takeaways

Remember:

  • : Creates links to other pages or resources
  • Relative URLs: For internal links within your site
  • Absolute URLs: For external links to other websites
  • target="_blank": Opens link in new tab
  • : Embeds images
  • alt attribute: Essential for accessibility
  • width/height: Prevents layout shift
