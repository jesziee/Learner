🔤 Lesson 1: Introduction to HTML

What HTML is and how it works

What is HTML?

HTML stands for HyperText Markup Language. It's the standard language used to create and structure content on the web. Every website you visit is built with HTML at its core.

Breaking down the name:
HyperText: Text that links to other text (like clicking links)
Markup: Tags that define how content is structured
Language: A standardized way to communicate instructions

HTML is Not a Programming Language

HTML is a markup language, not a programming language. It doesn't have logic, variables, or calculations. It simply describes the structure and content of a webpage.

// Programming languages (like JavaScript) can:
if (temperature > 30) {
    showMessage("It's hot!");
}

// HTML only describes structure:
<p>The temperature is 30 degrees.p>

Think of HTML as: The skeleton of a website. CSS adds the skin and clothes (style), and JavaScript adds the muscles and movement (behavior).

HTML Tags and Elements

HTML uses tags to mark up content. Tags tell the browser what each piece of content is.

Tag Syntax

// Basic tag structure:
<tagname>Content goes heretagname>
 ↑          ↑                    ↑
Opening    Content              Closing
tag                             tag

// Example:
<p>This is a paragraph.p>

// The opening and closing tags wrap the content.
// The closing tag always has a forward slash (/).

Self-Closing Tags

Some tags don't have content and close themselves:

// Self-closing tags (void elements):
<img src="photo.jpg" />    // Image
<br />                   // Line break
<hr />                   // Horizontal rule
<input type="text" />    // Input field

// In HTML5, the slash is optional:
<img src="photo.jpg">     // Also valid
<br>                    // Also valid

HTML Elements

An element includes the opening tag, content, and closing tag (or just the tag for self-closing elements).

// An element is the complete package:
┌─────────────────────────────────────┐
│ <h1>Welcome to My Websiteh1>    │
│ ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑ │
│       This entire line = Element     │
└─────────────────────────────────────┘

// Common elements you'll use:
<h1>Heading 1h1>           // Main heading
<p>Paragraph textp>       // Paragraph
<a>Click herea>           // Link
<div>Contentdiv>         // Container
<span>Inline textspan>     // Inline container

Nesting Elements

Elements can be placed inside other elements. This is called nesting.

// Nested elements:
<div>
    <h1>My Titleh1>
    <p>This is a <strong>boldstrong> word.p>
div>

// Correct nesting (tags close in reverse order):
<div><p>Textp>div>  ✓ // Correct

// Wrong nesting (tags don't match):
<div><p>Textdiv>p>  ✗ // Wrong!

Rule: Always close tags in the reverse order you opened them. Think of it like opening and closing parentheses in math.

HTML Attributes

Attributes provide extra information about elements. They go in the opening tag.

// Attribute syntax:
<tag attribute="value">Contenttag>

// Real examples:
<a href="https://google.com">Go to Googlea>
<img src="photo.jpg" alt="A beautiful sunset">
<p class="intro">Welcome!p>
<input type="text" placeholder="Enter name">

Common Attributes

  • id: Unique identifier for an element
  • class: Groups elements for styling
  • src: Source URL for images/media
  • href: URL for links
  • alt: Alternative text for images
  • style: Inline CSS styles

How HTML is Rendered

When you visit a website, here's what happens:

  1. The browser requests an HTML file from a server
  2. The server sends the HTML code
  3. The browser parses (reads) the HTML
  4. It builds a DOM (Document Object Model) tree
  5. It renders the visible page from the DOM

DOM: The Document Object Model is how the browser represents HTML as a tree structure. Each element is a "node" that JavaScript can interact with.
