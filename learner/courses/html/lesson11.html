What is SEO?

SEO (Search Engine Optimization) is the practice of improving your website so it ranks higher in search engine results like Google, Bing, and DuckDuckGo. Good SEO means more people can find your content organically — without paid ads. HTML plays a crucial role in SEO because search engines crawl and understand your HTML code.

1. The Title Tag

The title tag is one of the most important SEO elements. It appears in search results, browser tabs, and when sharing links on social media.




  
  Best Chocolate Chip Cookie Recipe | Sweet Bakery
Best practices for titles:
  • Length: 50-60 characters (Google truncates longer titles)
  • Keywords: Put important keywords near the beginning
  • Branding: Include your brand name after a pipe (|) or dash (-)
  • Unique: Each page should have a unique title

2. Meta Description

The meta description doesn't directly affect rankings, but it influences click-through rates from search results.
Line by line:
  • name="description" — Tells search engines this is the page description
  • content="..." — A compelling summary (150-160 characters ideal)
  • Includes target keywords naturally within a readable sentence
  • Encourages clicks with a clear value proposition

3. Heading Hierarchy (H1-H6)

Search engines use headings to understand your page structure. A proper hierarchy helps both SEO and accessibility.

Heading hierarchy rules:
  • One H1: Only one per page, describing the main topic
  • Logical order: Don't skip levels (H1 → H2 → H3, not H1 → H3)
  • Keywords: Include relevant keywords in headings naturally
  • Descriptive: Headings should summarize the content that follows

SEO benefits of semantic HTML:
  • — Indicates self-contained content that could be syndicated
  • — Helps search engines identify site structure
  • — Points to the primary content area
  • — Provides machine-readable date information

5. Image SEO

Images can appear in image search results and improve your overall SEO when optimized properly.
Image SEO checklist:
  • Filename: Use descriptive names with keywords (words separated by hyphens)
  • Alt text: Describe the image for accessibility and SEO
  • Dimensions: Include width/height to prevent layout shift
  • loading="lazy" — Defers loading off-screen images

6. Structured Data (Schema.org)

Structured data helps search engines understand your content and can create rich snippets in search results.
What this creates:
  • @type": "Recipe" — Tells Google this is a recipe
  • Enables rich results with cook time, ratings, and photos
  • Other types: Article, Product, LocalBusiness, FAQ, HowTo
  • Test with Google's Rich Results Test tool

7. Meta Tags for SEO

Additional meta tags provide information to search engines and social media platforms.
Meta tag purposes:
  • robots — Controls crawler behavior (index/noindex, follow/nofollow)
  • canonical — Specifies the preferred URL when duplicates exist
  • og:* — Controls how content appears when shared on social media
  • twitter:* — Twitter-specific card formatting

8. URL Structure

Clean, descriptive URLs help both users and search engines understand your page content.


https://example.com/p?id=123&cat=5
https://example.com/blog/post?a=1&b=2&c=3
https://example.com/this-is-a-very-long-url-with-too-many-keywords-stuffed-into-it


https://example.com/products/running-shoes
https://example.com/blog/how-to-bake-bread
https://example.com/recipes/chocolate-chip-cookies

9. Internal Linking

Linking between your own pages helps search engines discover content and understand site structure.



Learn more about bread baking basics 
before attempting this recipe. Check out our kneading 
tips for better results.





For more information, click here.




Read our complete baking guide for more information.





  Home
  Blog
  About
Internal linking best practices:
  • Descriptive anchor text: "bread baking basics" not "click here"
  • Relevant links: Link to related content
  • Reasonable number: Don't overload with links
  • aria-current="page" — Indicates current page in navigation

10. Mobile-First and Performance

Google uses mobile-first indexing, meaning it primarily uses the mobile version of your site for ranking.
Core Web Vitals: Google's Core Web Vitals (LCP, INP, CLS) are ranking factors. Test your pages with PageSpeed Insights to ensure good performance scores.

📚 Key Takeaways

  • Write unique, descriptive title tags (50-60 characters)
  • Include meta descriptions for better click-through rates
  • Use proper heading hierarchy (H1 → H2 → H3)
  • Semantic HTML helps search engines understand content
  • Optimize images with descriptive filenames and alt text
  • Use structured data for rich search results
  • Keep URLs clean and descriptive
  • Link internally with descriptive anchor text
