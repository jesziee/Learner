💿 Lesson 4: What is an Operating System?

Windows, macOS, Linux, and their roles

What is an Operating System?

An Operating System (OS) is the main software that manages all the hardware and software on your computer. It's the foundation that allows you to interact with your computer and run applications.

Think of the OS as: A traffic controller that manages all the "traffic" (data and programs) flowing through your computer, making sure everything gets to where it needs to go safely and efficiently.

What Does an OS Do?

The operating system performs many essential functions:

Memory Management

Decides how much RAM each program gets and ensures programs don't interfere with each other's memory.

Process Management

Decides which programs get CPU time and when. Your CPU can only do one thing at a time per core, so the OS rapidly switches between programs to create the illusion of multitasking.

File Management

Organizes and keeps track of all your files and folders on storage devices.

Device Management

Communicates with hardware devices through special programs called drivers. When you plug in a printer, the OS uses a driver to communicate with it.

User Interface

Provides the visual interface (windows, icons, menus) that lets you interact with your computer.

Security

Controls access to files and resources, requiring passwords and permissions to protect your data.

Key Point: Without an operating system, you'd have to write code to control every piece of hardware directly - even just to display a single character on screen!

Popular Operating Systems

Microsoft Windows

The most widely used desktop operating system in the world. Known for its compatibility with most software and hardware.

  • Used for: Gaming, business, general computing
  • Latest version: Windows 11
  • Strengths: Wide software compatibility, gaming support

macOS

Apple's operating system for Mac computers. Known for its elegant design and seamless integration with other Apple devices.

  • Used for: Creative work, development, productivity
  • Latest version: macOS Sonoma
  • Strengths: Design, ecosystem integration, Unix-based

Linux

An open-source operating system kernel that forms the basis for many different operating systems (called distributions or "distros").

  • Used for: Servers, development, privacy-conscious users
  • Popular distros: Ubuntu, Fedora, Debian, Arch
  • Strengths: Free, customizable, secure, runs on almost anything

Mobile Operating Systems

  • iOS: Apple's mobile OS for iPhones, known for security and app quality
  • Android: Google's mobile OS, based on Linux, highly customizable

Operating System Architecture

An OS is built in layers, each with specific responsibilities:

// OS Architecture Layers (from top to bottom):

┌─────────────────────────────────────┐
│        User Applications              │  ← Your programs
├─────────────────────────────────────┤
│        User Interface (GUI/Shell)        │  ← Windows, menus, commands
├─────────────────────────────────────┤
│        System Libraries                 │  ← Pre-built code for apps
├─────────────────────────────────────┤
│        System Services                   │  ← Background tasks
├─────────────────────────────────────┤
│        KERNEL                          │  ← Core OS functions
├─────────────────────────────────────┤
│        Device Drivers                   │  ← Hardware communication
├─────────────────────────────────────┤
│        Hardware                         │  ← Physical components
└─────────────────────────────────────┘

The Kernel

The kernel is the core of the operating system. It has complete control over everything and manages:

  • Memory allocation
  • CPU scheduling
  • Device communication
  • System security

File Systems

Operating systems organize data using file systems - structures that determine how files are stored, named, and accessed.

Common File Systems

  • NTFS: Windows default, supports large files and security features
  • APFS: Apple's modern file system for macOS and iOS
  • ext4: Common Linux file system, reliable and efficient
  • FAT32/exFAT: Compatible with most systems, used for USB drives

File Paths

Files are located using paths - addresses that show where a file is stored:

// Windows path:
C:\Users\YourName\Documents\report.txt

// macOS/Linux path:
/home/yourname/documents/report.txt

// Key differences:
// - Windows uses backslashes (\) and drive letters (C:)
// - macOS/Linux use forward slashes (/) and start from root (/)

Running Programs

When you double-click an application, here's what happens:

  1. Loading: The OS reads the program from storage into RAM
  2. Memory Allocation: Space is assigned for the program's data
  3. Execution: The CPU starts running the program's instructions
  4. Resource Management: The OS provides access to screen, keyboard, network as needed
  5. multitasking: The OS shares CPU time with other running programs

Background Processes: Your computer is always running dozens of processes even when you're not actively using applications. These handle system tasks like checking for updates, managing network connections, and more.

Key Takeaways

Remember:

  • OS: The main software that manages hardware and runs applications
  • Kernel: The core of the OS with direct hardware access
  • Main OS types: Windows, macOS, Linux (desktop); iOS, Android (mobile)
  • File System: How the OS organizes and stores data
  • Processes: Programs that are currently running in memory
