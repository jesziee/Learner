Lesson 6: Functions

Master the art of creating reusable code blocks with functions.

What are Functions?

Functions are reusable blocks of code that perform specific tasks. Think of them as mini-programs within your program. Instead of writing the same code multiple times, you write it once as a function and call it whenever needed. This makes your code cleaner, more organized, and easier to maintain.

Why Functions Matter: Functions are the building blocks of any program. They help you write DRY code (Don't Repeat Yourself), organize your logic into manageable pieces, and make your code easier to test and debug.

Function Declaration

The traditional way to create a function:

// Basic function declaration
function greet() {
    console.log("Hello, World!");
}

// Call (invoke) the function
greet();  // Output: Hello, World!
greet();  // Output: Hello, World!
greet();  // Output: Hello, World!
Function Parts:
  • function - Keyword that declares a function
  • greet - Function name (use descriptive names)
  • () - Parameters go here (empty if no parameters)
  • { } - Function body contains the code to execute
  • greet(); - Calling/invoking the function

Parameters and Arguments

Pass data into functions to make them dynamic:

// Function with parameters
function greetPerson(name) {
    console.log("Hello, " + name + "!");
}

// Call with different arguments
greetPerson("Alice");    // Output: Hello, Alice!
greetPerson("Bob");      // Output: Hello, Bob!
greetPerson("Charlie");  // Output: Hello, Charlie!
Parameters vs Arguments:
  • Parameter: Variable in the function definition (name)
  • Argument: Actual value passed when calling ("Alice")
  • Parameters act as placeholders for the values you'll pass in

Multiple Parameters

// Function with multiple parameters
function introduce(name, age, city) {
    console.log("Hi, I'm " + name);
    console.log("I'm " + age + " years old");
    console.log("I live in " + city);
}

introduce("John", 25, "New York");

// Output:
// Hi, I'm John
// I'm 25 years old
// I live in New York

Default Parameters

// Parameters with default values
function greetWithDefault(name = "Guest") {
    console.log("Hello, " + name + "!");
}

greetWithDefault();         // Output: Hello, Guest!
greetWithDefault("Alice");  // Output: Hello, Alice!

// Multiple default parameters
function createUser(name = "Anonymous", role = "user", active = true) {
    console.log(`Name: ${name}, Role: ${role}, Active: ${active}`);
}

createUser();                      // Name: Anonymous, Role: user, Active: true
createUser("Admin", "admin");      // Name: Admin, Role: admin, Active: true
createUser("Bob", "editor", false); // Name: Bob, Role: editor, Active: false

Return Values

Functions can send values back to the caller:

// Function that returns a value
function add(a, b) {
    return a + b;
}

// Store the returned value
let result = add(5, 3);
console.log(result);  // Output: 8

// Use directly in expressions
console.log(add(10, 20));        // Output: 30
console.log(add(2, 3) * 2);      // Output: 10
console.log("Sum: " + add(7, 8)); // Output: Sum: 15
How Return Works:
  • return - Keyword that sends a value back
  • Function stops executing when it hits return
  • Without return, functions return undefined
  • Returned values can be stored, displayed, or used in calculations

Early Return

// Return exits the function immediately
function divide(a, b) {
    if (b === 0) {
        return "Cannot divide by zero";
    }
    return a / b;
}

console.log(divide(10, 2));   // Output: 5
console.log(divide(10, 0));   // Output: Cannot divide by zero

// Multiple return points
function getGrade(score) {
    if (score >= 90) return "A";
    if (score >= 80) return "B";
    if (score >= 70) return "C";
    if (score >= 60) return "D";
    return "F";
}

console.log(getGrade(85));  // Output: B

Function Expressions

Store functions in variables:

// Function expression
const sayHello = function() {
    console.log("Hello!");
};

sayHello();  // Output: Hello!

// Function expression with parameters
const add = function(a, b) {
    return a + b;
};

console.log(add(3, 4));  // Output: 7

// Named function expression (useful for debugging)
const multiply = function multiplyNumbers(a, b) {
    return a * b;
};

console.log(multiply(3, 4));  // Output: 12
Declaration vs Expression:
  • Declaration: function name() { } - Hoisted (can call before definition)
  • Expression: const name = function() { } - Not hoisted
  • Expressions are often preferred in modern JavaScript

Arrow Functions

A modern, concise way to write functions (ES6+):

// Traditional function
const addTraditional = function(a, b) {
    return a + b;
};

// Arrow function
const addArrow = (a, b) => {
    return a + b;
};

// Arrow function with implicit return
const addShort = (a, b) => a + b;

console.log(addTraditional(2, 3));  // 5
console.log(addArrow(2, 3));       // 5
console.log(addShort(2, 3));       // 5
Arrow Function Syntax:
  • (params) => { body } - Full syntax
  • param => { body } - Single parameter (no parens needed)
  • (params) => expression - Implicit return (no braces)
  • () => expression - No parameters

Arrow Function Variations

// No parameters
const sayHi = () => console.log("Hi!");
sayHi();  // Hi!

// One parameter (parentheses optional)
const double = x => x * 2;
console.log(double(5));  // 10

// Multiple parameters
const subtract = (a, b) => a - b;
console.log(subtract(10, 3));  // 7

// Multiple lines (need braces and return)
const calculate = (a, b, operation) => {
    if (operation === "add") return a + b;
    if (operation === "subtract") return a - b;
    return 0;
};

// Returning an object (wrap in parentheses)
const makePerson = (name, age) => ({ name, age });
console.log(makePerson("Alice", 25));  // { name: "Alice", age: 25 }

Scope and Functions

Variables have different visibility depending on where they're declared:

// Global scope
let globalVar = "I'm global";

function myFunction() {
    // Function scope
    let localVar = "I'm local";
    
    console.log(globalVar);  // Accessible
    console.log(localVar);   // Accessible
    
    // Block scope inside function
    if (true) {
        let blockVar = "I'm in a block";
        console.log(blockVar);  // Accessible
    }
    
    // console.log(blockVar);  // Error! Not accessible outside block
}

myFunction();
console.log(globalVar);   // Accessible
// console.log(localVar); // Error! Not accessible outside function
Scope Types:
  • Global: Accessible everywhere (avoid if possible)
  • Function: Accessible only within the function
  • Block: Accessible only within { } block (let/const)

Closures

Functions can "remember" variables from their outer scope:

// A closure example
function createCounter() {
    let count = 0;  // Private variable
    
    return function() {
        count++;
        return count;
    };
}

const counter = createCounter();
console.log(counter());  // 1
console.log(counter());  // 2
console.log(counter());  // 3

// count is "enclosed" in the returned function
// console.log(count);  // Error! count is not accessible

// Practical example: Configuration
function createGreeter(greeting) {
    return function(name) {
        return `${greeting}, ${name}!`;
    };
}

const sayHello = createGreeter("Hello");
const sayHi = createGreeter("Hi");

console.log(sayHello("Alice"));  // Hello, Alice!
console.log(sayHi("Bob"));        // Hi, Bob!
Why Closures Matter:
  • Create private variables
  • Maintain state between function calls
  • Create configurable functions
  • Essential for callbacks and event handlers

Higher-Order Functions

Functions that take other functions as arguments or return functions:

// Function that takes a function as argument
function applyOperation(a, b, operation) {
    return operation(a, b);
}

const add = (x, y) => x + y;
const multiply = (x, y) => x * y;

console.log(applyOperation(5, 3, add));      // 8
console.log(applyOperation(5, 3, multiply)); // 15

// Built-in higher-order functions
const numbers = [1, 2, 3, 4, 5];

// map - transform each element
const doubled = numbers.map(n => n * 2);
console.log(doubled);  // [2, 4, 6, 8, 10]

// filter - keep elements that pass a test
const evens = numbers.filter(n => n % 2 === 0);
console.log(evens);  // [2, 4]

// reduce - combine elements into one value
const sum = numbers.reduce((total, n) => total + n, 0);
console.log(sum);  // 15

Rest Parameters

Accept any number of arguments:

// Rest parameters (...args)
function sumAll(...numbers) {
    let total = 0;
    for (let num of numbers) {
        total += num;
    }
    return total;
}

console.log(sumAll(1, 2, 3));           // 6
console.log(sumAll(1, 2, 3, 4, 5));     // 15
console.log(sumAll(1, 2, 3, 4, 5, 6));  // 21

// Rest with regular parameters
function logItems(category, ...items) {
    console.log("Category:", category);
    console.log("Items:", items);
}

logItems("Fruits", "Apple", "Banana", "Orange");
// Category: Fruits
// Items: ["Apple", "Banana", "Orange"]
Rest Parameter Syntax:
  • ...name - Collects remaining arguments into an array
  • Must be the last parameter in the function
  • Allows flexible number of arguments

Callback Functions

Functions passed to other functions to be called later:

// Basic callback
function processData(data, callback) {
    console.log("Processing:", data);
    const result = callback(data);
    console.log("Result:", result);
}

processData(5, x => x * 2);
// Processing: 5
// Result: 10

// Async callback (simulated)
function fetchUser(userId, onSuccess, onError) {
    // Simulating async operation
    setTimeout(() => {
        if (userId > 0) {
            onSuccess({ id: userId, name: "User " + userId });
        } else {
            onError("Invalid user ID");
        }
    }, 1000);
}

fetchUser(
    1,
    user => console.log("Success:", user),
    error => console.log("Error:", error)
);
// After 1 second: Success: { id: 1, name: "User 1" }

IIFE (Immediately Invoked Function Expression)

Functions that run immediately after being defined:

// IIFE pattern
(function() {
    console.log("This runs immediately!");
})();

// IIFE with arrow function
(() => {
    console.log("Arrow IIFE");
})();

// IIFE with parameters
(function(name) {
    console.log("Hello, " + name);
})("World");
// Output: Hello, World

// Useful for creating private scope
const counter = (function() {
    let count = 0;
    return {
        increment: () => ++count,
        decrement: () => --count,
        getCount: () => count
    };
})();

console.log(counter.increment());  // 1
console.log(counter.increment());  // 2
console.log(counter.getCount());   // 2

Practical Examples

// Example 1: Temperature converter
const toCelsius = fahrenheit => (fahrenheit - 32) * 5/9;
const toFahrenheit = celsius => (celsius * 9/5) + 32;

console.log(toCelsius(68));     // 20
console.log(toFahrenheit(20));  // 68

// Example 2: Password validator
function validatePassword(password) {
    const checks = {
        length: password.length >= 8,
        hasUpper: /[A-Z]/.test(password),
        hasLower: /[a-z]/.test(password),
        hasNumber: /[0-9]/.test(password)
    };
    
    const valid = Object.values(checks).every(c => c);
    return { valid, checks };
}

console.log(validatePassword("Pass123!"));

// Example 3: Curry function
function multiply(a) {
    return function(b) {
        return function(c) {
            return a * b * c;
        };
    };
}

console.log(multiply(2)(3)(4));  // 24

const multiplyBy2 = multiply(2);
const multiplyBy2And3 = multiplyBy2(3);
console.log(multiplyBy2And3(5));  // 30

Practice Exercise

// Exercise: Build a shopping cart system
const cart = [];

function addItem(name, price, quantity = 1) {
    cart.push({ name, price, quantity });
}

function removeItem(name) {
    const index = cart.findIndex(item => item.name === name);
    if (index > -1) {
        cart.splice(index, 1);
    }
}

function getTotal() {
    return cart.reduce((total, item) => total + (item.price * item.quantity), 0);
}

function printCart() {
    console.log("=== Shopping Cart ===");
    cart.forEach(item => {
        console.log(`${item.name}: $${item.price} x ${item.quantity}`);
    });
    console.log(`Total: $${getTotal().toFixed(2)}`);
}

// Test the cart
addItem("Apple", 1.50, 3);
addItem("Bread", 2.99);
addItem("Milk", 3.49);
printCart();

removeItem("Bread");
printCart();
