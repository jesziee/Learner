Lesson 1: Introduction to JavaScript

Learn what JavaScript is, where it's used, and write your first script!

What is JavaScript?

JavaScript is a programming language that brings websites to life. While HTML provides structure and CSS adds style, JavaScript adds interactivity and dynamic behavior. It's one of the most popular programming languages in the world and runs in every modern web browser.

Fun Fact: JavaScript was created in just 10 days by Brendan Eich in 1995. Despite its name, JavaScript has nothing to do with Java - they are completely different languages!

Where is JavaScript Used?

JavaScript is incredibly versatile and is used in many areas:

  • Websites: Interactive forms, animations, and user interfaces
  • Web Applications: Google Maps, Gmail, Facebook
  • Mobile Apps: React Native, Ionic frameworks
  • Desktop Apps: VS Code, Slack, Discord
  • Server-side: Node.js for backend development
  • Games: Browser-based games and game engines

Your First JavaScript Program

Let's write a simple program that greets the user:

// This is our first JavaScript program
// It will ask for your name and greet you

let userName = prompt('What is your name?');
alert('Hello, ' + userName + '! Welcome to JavaScript!');
Line by Line Explanation:
  • // This is our first JavaScript program - A comment that explains what the code does (JavaScript ignores this)
  • let userName - Creates a variable (container) to store data
  • = - Assignment operator, puts a value into the variable
  • prompt('What is your name?') - Shows a dialog asking for user input
  • alert('Hello, ' + userName + '!') - Shows a greeting using the stored name
  • + - Combines (concatenates) text strings together

Opening the Browser Console

The browser console is where you can see JavaScript output and errors, and even write JavaScript directly:

How to Open the Console:

  • Chrome/Edge: Press F12 or Ctrl+Shift+J (Cmd+Option+J on Mac)
  • Firefox: Press F12 or Ctrl+Shift+K (Cmd+Option+K on Mac)
  • Safari: Enable Developer menu in Preferences, then Cmd+Option+C
// Try typing this in the console:
console.log('Hello, Console!');
console.log(2 + 2);
console.log('JavaScript is awesome!');
Code Explanation:
  • console.log() - Prints output to the console (useful for debugging)
  • 2 + 2 - JavaScript can do math! This outputs 4
  • Each console.log() creates a new line of output

JavaScript Statements

A JavaScript statement is an instruction that the browser executes. Statements are usually separated by semicolons:

// Each line is a statement
let x = 5;           // Statement 1: Create variable
let y = 10;          // Statement 2: Create another variable
let sum = x + y;     // Statement 3: Add them together
console.log(sum);    // Statement 4: Display the result
Code Explanation:
  • let x = 5; - Creates variable x with value 5
  • let y = 10; - Creates variable y with value 10
  • let sum = x + y; - Creates sum with value 15 (5 + 10)
  • ; - Semicolon marks the end of a statement
Best Practice: Always end your statements with semicolons. While JavaScript can sometimes work without them, using semicolons prevents errors and makes your code clearer.

JavaScript is Case-Sensitive

JavaScript treats uppercase and lowercase letters as different characters:

// These are three different variables:
let myVariable = 1;
let MyVariable = 2;
let MYVARIABLE = 3;

console.log(myVariable);   // Outputs: 1
console.log(MyVariable);   // Outputs: 2
console.log(MYVARIABLE);   // Outputs: 3
Code Explanation:
  • myVariable, MyVariable, MYVARIABLE - All different variables
  • Capitalization matters in JavaScript!
  • This applies to variables, functions, and keywords

Practice Exercise

Try writing JavaScript that:

  1. Opens the browser console (F12)
  2. Types console.log('My name is [your name]');
  3. Creates a variable for your age: let age = [your age];
  4. Displays it: console.log('I am ' + age + ' years old');
Remember: Practice is key! The more you write JavaScript, the more natural it becomes. Don't be afraid to experiment and make mistakes - that's how we learn!
